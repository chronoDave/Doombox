# Doombox - Keybinds

## Windows

| System (non-configurable) |  |
|:------------------ | ---------- |
| **Ctrl + =**       | Zoom in    |
| **Ctrl + -**       | Zoom out   |
| **Ctrl + Alt + =** | Reset zoom |

| File | |
|:-------------------- | ---------------- |
| **Ctrl + S**         | Rescan library   |
| **Ctrl + Shift + S** | Scan folder      |
| **Ctrl + Shift + I** | Toggle dev tools |

| Playlist | |
| :------------------ | ------------- |
| **Ctrl + Alt + →**  | Next song     |
| **Ctrl + Alt + ←**  | Previous song |
| **Ctrl + Alt + \\** | Play / pause  |

## Mac

| System (non-configurable) | |
|:------------------ | ---------- |
| **⌘ + =**         | Zoom in    |
| **⌘ + -**         | Zoom out   |
| **⌘ + ⌥ + =**     | Reset zoom |

| File | |
|:-------------------- | ---------------- |
| **⌘ + S**           | Rescan library   |
| **⌘ + ⇧ + S**       | Scan folder      |
| **⌘ + ⇧ + I**       | Toggle dev tools |

| Playlist | |
| :--------------- | ------------- |
| **⌘ + ⌥ + →**  | Next song     |
| **⌘ + ⌥ + ←**  | Previous song |
| **⌘ + ⌥ + \\** | Play / pause  |
